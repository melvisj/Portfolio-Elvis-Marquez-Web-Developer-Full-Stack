Mis Certificaciones

Estos son los cursos que he completado en Coderhouse. Cada uno representa una etapa importante en mi desarrollo como profesional en el Desarrollo Web.

Actualmente sigo creciendo, aprendiendo nuevas tecnologias y perfeccionando las ya adquiridas.

Este curso intensivo de 10 semanas me permitió adquirir una base sólida en el desarrollo web moderno. A través de clases en vivo y desafíos prácticos, aprendí a construir sitios web desde cero utilizando HTML y CSS, aplicando principios de diseño responsivo, buenas prácticas de accesibilidad y optimización básica para motores de búsqueda (SEO).

Esta formación fue el punto de partida para seguir especializándome en tecnologías frontend y backend, siempre con una visión práctica y enfocada en las necesidades reales del mundo digital.

Este curso me permitió dominar los fundamentos de la programación con JavaScript, el lenguaje que da vida a la web. Durante la formación, aprendí a manipular el DOM, trabajar con eventos, desarrollar scripts interactivos y aplicar buenas prácticas de desarrollo.

También adquirí nociones clave sobre asincronía (promesas, async/await) y consumo de APIs, lo cual me permite crear interfaces dinámicas y conectadas a servicios externos. Fue un paso esencial para afianzar mi perfil como desarrollador frontend, con una base sólida en lógica de programación y experiencia en proyectos prácticos.

En este curso aprendí a desarrollar aplicaciones modernas con React, el framework más popular del ecosistema frontend. Comprendí la lógica basada en componentes, el uso de props y estados, y cómo gestionar el ciclo de vida de las interfaces.

También adquirí experiencia con herramientas como React Router para crear navegación entre vistas, y Context para el manejo eficiente del estado global.

Gracias a una metodología práctica centrada en proyectos reales, salí preparado para construir aplicaciones escalables, mantenibles y con una experiencia de usuario fluida y profesional.

Este curso me permitió comprender a fondo la lógica del lado del servidor, estructurando aplicaciones robustas, eficientes y seguras. Aprendí a trabajar con Node.js, Express y bases de datos como MongoDB, implementando rutas, middlewares, autenticación, y controladores bajo buenas prácticas de arquitectura.

Durante el proceso desarrollé proyectos escalables, utilizando herramientas como Postman para testing, Git para control de versiones y principios de MVC para mantener el código limpio y modular. Esta formación fortaleció mi capacidad de construir APIs sólidas, integraciones con front-end y soluciones enfocadas en rendimiento y seguridad.
